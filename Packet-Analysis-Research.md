# 🛡️ Network Packet Analysis & Protocol Research

### 📊 Objective
Understanding the fundamental behavior of network traffic and identifying security gaps in communication protocols.

### 🔍 Research Topics
1. **The 3-Way Handshake (TCP/IP)**
   - Analysis of SYN, SYN-ACK, and ACK packets.
   - Why it matters for connection reliability and security.

2. **Protocol Vulnerability Assessment**
   - Examining unencrypted protocols (HTTP, FTP, Telnet).
   - Practical analysis of data interception risks.

3. **Traffic Visualization**
   - Using Wireshark to decode complex packet structures.
   - Identifying unusual patterns that indicate network scanning or attacks.

### 🛠️ Environment
- **OS:** Kali Linux / Parrot Security
- **Tools:** Wireshark, TCPDump, Nmap
- **Network:** Private Lab Environment
